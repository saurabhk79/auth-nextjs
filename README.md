# Next auth assignment

A simple nextjs login and password authentication system with redux


## Login Data

```http
  GET /api/auth/login
```

| User | Password     | 
| :-------- | :------- |
| `user` | `00000000` |
| `test_user` | `12345678` |
| `myuser` | `12345678` |



## Installation

Install my-project with npm

```bash
  npm install auth-nextjs
  cd auth-nextjs
  npm run dev
```
